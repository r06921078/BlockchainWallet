BlockchainWallet
=================

**BlockchainWallet** use the testnet *(testchain)* , and all of UI is command line.  
And is a wallet of blockchain by call APIs.

* Requirement

> python3  
> block-io == (1.1.6)


* Three coin in testnet can be used.

> Bitcoin  
> Litecoin  
> Dogecoin
